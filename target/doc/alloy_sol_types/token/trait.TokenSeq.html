    const IS_TUPLE: bool = false;

    // Required methods
    fn encode_sequence(&self, enc: &mut Encoder);

}
Expand description

A token composed of a sequence of other tokens

This functions as an extension trait for TokenType, and may only be implemented by FixedSeqToken, DynSeqToken, and PackedSeqToken.

Provided Associated Constants§

source
