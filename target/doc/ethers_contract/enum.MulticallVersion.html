Enum ethers_contract::MulticallVersion

#[repr(u8)]
pub enum MulticallVersion {
    Multicall,
    Multicall2,
    Multicall3,
}
Expand description

The version of the Multicall. Used to determine which methods of the Multicall smart contract to use:

  • Multicall : aggregate((address,bytes)[])
  • Multicall2 : try_aggregate(bool, (address,bytes)[])
  • Multicall3 : aggregate3((address,bool,bytes)[]) or aggregate3Value((address,bool,uint256,bytes)[])
